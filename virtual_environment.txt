conda create -n <environment_name> python=3.8 - создать новое виртуальное окружение
conda info -e - посмотреть доступные окружения
conda activate <environment_name> - запустить нужное окружение
conda install -c conda-forge jupyter - загрузить jupyter в текущее окружение
jupyter notebook - запустить jupyter notebook
conda install -c conda-forge <lib_name> - загрузить библиотеку (работает не всегда)
conda list -e > requirements.txt - создание requirements
